What is the difference between Margin, Border and Padding?

Margin: The margin creates space around elements, essentially pushes away its neighbors. CSS has properties for specifying the margin for each side of an element: margin-top, margin-bottom, margin-left, margin-right.

Border: Border is the strip that wraps around elements. The style, width and color of the border can be specified with the CSS border properties. e.g. dotted, dashed, solid, etc.

Padding: Unlike margins and borders, padding is located inside the border of an element. An element's padding is the space between its contenct and its border, which creates space within an element.

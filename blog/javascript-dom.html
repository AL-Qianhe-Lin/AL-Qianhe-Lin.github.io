Explain control flow and loops using an example process from everyday life

Control flow in Javasciprt is how the order that computer runs code, from top to bottom. Loops repeat a series of actions under certain conditions. Take cooking as an example:

1. Prepare ingridients
2. Wash and cut ingridients
3. Cook everything
4. Serve!

The loop can be run everything cooking happens, and conditions can be specified. For example, if soy sauce runs out, that certain action cannot happen, there needs to be other events taking place like go buy some or change the recipe.

Describe what the DOM is and an example of how you might interact with it.

The DOM(Document Object Model) is essencially a tree of Objects. DOM enables Javascript the power to make lots of changes such as HTML elements, attributes, CSS styles, etc.

Explain the difference between accessing data from arrays and objects.

Objects represents a 'thing' in a code, which could be anything. It can store a set of characteristics called properties, which are made up of a key and a value. Properties in objects can be accessed using either dot of bracket.

Arrays also store a list of items, however, data in arrays are accessed by their numerical position in the list, with zero-based indexing, where the first item has the position zero, and so on.

Explain what functions are and why they are helpful.

A function is a block of code, made to perform a particualr task. A function is executed when it is triggered(called). Functions are helpful because codes can then be reused. Once the code is defined with a function, it can be used many times with different parameters, thus produce different results.
